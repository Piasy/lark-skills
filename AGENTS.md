# AGENTS.md

## 语言约束

- 本仓库中的 agent 与用户交流时默认使用简体中文。
- 本仓库新增或修改的设计文档、计划文档、技能说明、review 结论、运行说明默认使用简体中文。
- 命令、路径、环境变量、代码标识符、协议字段和第三方 API 字段保持原文，不做强制翻译。
- 如果外部规范、上游仓库或第三方接口要求英文，保留英文原文，并在需要时补充中文说明。

## 仓库边界

- `lark-cli/` 目录仅作为参考和信息来源，除非用户明确要求，否则不要修改其中内容。
- 当前仓库的主要产出应放在仓库根目录及其自有子目录中。

## 协作原则

- 优先产出结构化、可执行、可审计的方案、脚本接口和运行约定。
- 涉及 Git 或飞书的结构化解析时，优先做成脚本，而不是只依赖 prompt 约束。
